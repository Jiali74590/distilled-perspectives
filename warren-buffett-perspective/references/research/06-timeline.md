# 06 - 巴菲特人物时间线

## 早期 1930-1956（形成期）

| 时间 | 事件 | 影响 |
|------|------|------|
| 1930.8.30 | 生于Omaha | 起点。父亲Howard是股票经纪人+后来国会议员 |
| 1936 | 6岁开始卖Coke、口香糖 | 商业本能启蒙 |
| 1941 | 11岁第一次买股票（Cities Service Preferred） | 过早尝试 |
| 1942 | 父亲当选国会议员，全家搬Washington DC | 离开Omaha初体验 |
| 1944 | 14岁报税（报童收入$592）| 早熟的纳税申报——巴菲特后来反复提这个 |
| 1947-1949 | Wharton (Univ of Pennsylvania) 商学院 | 不喜欢Wharton，觉得教授不如他懂得多 |
| 1949-1950 | 转Nebraska University | 读完学士 |
| 1950 | 被Harvard Business School拒绝 | [转折#1] 这让他去Columbia找到Graham——命运转折 |
| 1950-1951 | Columbia Business School 读Graham | 第一次接触value investing，Graham给他唯一的A+ |
| 1951-1954 | 回Omaha做stockbroker | 挣扎期，Graham让他读书 |
| 1954-1956 | Graham-Newman Partnership | [转折#2] 直接跟Graham学习 |
| 1956 | Graham关闭partnership，Buffett回Omaha | 开始自己的Buffett Partnership |

## Partnership时代 1956-1969（deep value阶段）

| 时间 | 事件 | 影响 |
|------|------|------|
| 1956 | 成立Buffett Partnership（$105K，家人朋友钱） | Graham-style operations开始 |
| 1957-1969 | Partnership年化29.5% vs 道指7.4% | 惊人业绩奠定声誉 |
| 1959 | 遇Charlie Munger | [转折#3] 但两人合作要到1970s才真正开始 |
| 1962 | 开始买入Berkshire Hathaway（纺织公司） | [战略错误] 他后来说这是最大的mistake of first 25 years |
| 1965 | 控股Berkshire Hathaway | 用它作为investment vehicle |
| 1967 | 买入National Indemnity（$8.6M） | [转折#4] 保险业务起点——float帝国的种子 |
| 1969 | 关闭Buffett Partnership | [转折#5] 市场估值太高找不到机会，"这不是我能玩的游戏了" |

## 过渡期 1970-1988（从Graham到Munger）

| 时间 | 事件 | 影响 |
|------|------|------|
| 1970 | Berkshire成为主要investment vehicle | 身份从partner到CEO |
| 1972 | See's Candies通过Blue Chip Stamps收购（$25M） | [转折#6] 第一次买great business——Munger坚持 |
| 1973 | Washington Post投资（$10.6M） | 与Katharine Graham终身友谊 |
| 1976 | GEICO濒临破产，Buffett买入 | float基地扩张 |
| 1977 | Fortune essay "How Inflation Swindles the Equity Investor" | 思想成型 |
| 1977 | Susan搬去San Francisco独居 | [私生活转折] Astrid Menks 1978年起和Buffett住一起 |
| 1983 | 股东信首次系统讲economic goodwill | 为great business估值奠基 |
| 1986 | 股东信定义"Owner Earnings" | 非GAAP现金流概念成型 |
| 1987 | 股东信详细讲Mr. Market寓言 | 经典营销 |
| 1988 | **开始建仓Coca-Cola（$1B+）** | [转折#7] great business阶段正式开始 |

## 黄金时代 1989-2008（great business阶段）

| 时间 | 事件 | 影响 |
|------|------|------|
| 1989 | 股东信《Mistakes of the First 25 Years》 | 系统化自我反省 |
| 1991 | **Salomon丑闻 + 国会听证** | [转折#8] "lose reputation and I'll be ruthless" 公众形象定型 |
| 1995-96 | 完全收购GEICO（$2.3B） | float帝国巩固 |
| 1998 | General Re收购（$22B全股换购） | [战略错误] 他后来称最差acquisition，但给了Ajit Jain舞台 |
| 1999 | Allen & Co Sun Valley演讲 | 预警科技股泡沫——事后被证明正确 |
| 2000 | dotcom泡沫破灭 | 他没参与涨，也就没受伤——逆周期声誉建立 |
| 2001 | 9/11后保险巨亏 | 承认对极端事件定价不足 |
| 2006 | **$62万拍卖巴菲特午餐，中标者段永平+黄峥** | 跨文化影响延伸到中国 |
| 2006.6 | 宣布捐99%财产给慈善（主要Gates Foundation, $30B+承诺） | 巨额捐赠承诺——但注意他没退休 |
| 2008.9 | **Goldman Sachs $5B + GE $3B preferred deal** | 危机中的crisis-privileged投资 |
| 2008.10.16 | **NYT op-ed "Buy American. I Am."** | 市场最黑暗时公开买入 |

## 晚期转型 2011-2024

| 时间 | 事件 | 影响 |
|------|------|------|
| 2011 | 买入IBM $10.7B | [转折#9] 第一次大规模买tech——后来证明错 |
| 2016 | **Apple初始建仓（Ted Weschler主导）** | [转折#10] tech的第二次尝试——这次对，但不是Buffett自己的idea |
| 2017 | Becoming Warren Buffett (HBO) | 公众形象更柔软 |
| 2018 | 完全退出IBM、大幅加仓Apple | 承认IBM错 |
| 2019 | Apple成最大持仓 | tech成为Berkshire最大exposure |
| 2020.3-5 | **疫情期间卖airlines + 没做大动作** | 最受批评的period——保留大量cash没买 |
| 2020.8 | **买入日本五大商社** | 最成功的晚年call之一 |
| 2021.5 | Munger在年会口误泄露"Greg Abel will keep the culture" | 继承人问题首次半公开 |
| 2022.3 | Occidental Petroleum大举建仓（14%两周内） | 能源回归 |
| 2023.11.28 | **Charlie Munger去世（99岁）** | [转折#11] 64年合作终结 |
| 2024.2 | 股东信怀念Munger"the architect of Berkshire" | Munger时代official结束 |
| 2024.5 | 年会宣布Greg Abel年底接任CEO | [转折#12] 正式succession |
| 2024全年 | **Apple持仓减持75%，现金创$334B纪录** | 巨大rebalancing |

## 最后阶段 2025-2026

| 时间 | 事件 | 影响 |
|------|------|------|
| 2025.5 | 最后一次以CEO身份主持年会 | 告别情绪浓厚 |
| 2025.11.10 | **Thanksgiving letter宣布"going quiet"** | 最后一次正式letter；Abel将2026.1.1接任；他继续写Thanksgiving letter但不再写annual report |
| 2026.1.1 | **Greg Abel正式成为Berkshire CEO** | Buffett 60年CEO时代结束；他继任Chairman（但真正control给Abel） |

| 2026.4（本skill调研时间） | 95岁+，仍然阅读6小时/天，在Omaha家中 | 长寿+复利的活样本 |
